TableOS: CloudTracker PoC

1. Using a Nodejs server („cloud“) to store data in a SQLite DB in the format of JSON.
1.1 Authentication: login or register with username and password to get token for cloud memorization
2. Using an Android app to track GPS coordinates inside the cloud.
3. Using a HTML site with Javascript using Leaflet to visualize the coordinates.

Implementation

- Vibe coded through Claude
- Raspberry Pi 4 is used to serve cloud
- Android Studio is used to build Android app
